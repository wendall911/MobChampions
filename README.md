NeoForge / Fabric Mod that provides a hybrid of Champions and Mob Captains.

Elite mobs can spawn with different levels, abilities and enhanced stats. Champions
drop rare loot and unobtainable items.

Highly configurable alternative to Champions / Mob Captains / Infernal Mobs
mods for modern versions of Minecraft.
